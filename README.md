# Playwright-QA-Practice
This is a repository for my learning exercises of Playwright on qa-practice.com
Each file that ends with _test.py is a set of test cases written by me for different features on the website.
I've set the CI for the QA Practice site, so if there was a Webhook linking the qa-practice.com development repository to mine, whenever they would push code that modifies the website, my repository would be pinged, the tests would start and if anything would fail, they would be blocked from pushing the code until all the tests have passed.
